Projects:

  • 4th Grade Math Practice Problems. Written in python, this code generates a random question from one of two different problem types and presents it to the user to solve.   • Bug Sharing & Developer Communication Platform. Built using code.org's AppLab, this allows developers to have a centralized space for users to report bugs. Version 1.0 was submitted to the 2023 AP CSP Exam. A.P. Computer Science Feedback/Bug Reports
